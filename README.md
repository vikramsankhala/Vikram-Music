# Home Band & Jam Room

Complete equipment & wiring guide for building a professional home jam room — PA speakers, mixer, electronic drums, guitar rig, Ableton Live integration, fusion music, and Pune sourcing.

## Contents

- **website/** — Full static site ready for Netlify deployment
- **Home Jam Room.docx** — Source document with equipment specs and cost estimates

## Quick Start

1. Open `website/index.html` in a browser, or
2. Deploy the `website` folder to [Netlify](https://netlify.com)

See [website/README.md](website/README.md) for deployment details.
